# Beer.js

Add beer to your js

## Idea

Based on [beers.rb](https://github.com/mutru/beers)

## Usage

### Demo

`$ node demo.js` or [demo](https://michaeldanilov.github.io/beer.js/)

### API

It extends Number and Date JavaScript's data type.
```js
// require and init
require('./beer').bar();

// get 1 beer
(1).beer();

// get 2 beers
(2).beer();

// get double beers
(1).doubleBeers();

// get a beer pack. which contains 6 beers
(6).beerPack();

// check beertime
(new Date).isBeerTime();
```
